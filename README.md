# CV Web — Argenis Zamora

Mi currículum personal hecho con **React + Vite**, con tema oscuro/tech y
publicado automáticamente en **GitHub Pages**.

🔗 **En vivo:** https://argeniszamora.github.io

## ✏️ Cómo editar mi información

Todo el contenido del CV está en un solo archivo:

```
src/cvData.js
```

Ahí editas tu nombre, experiencia, habilidades, proyectos, etc.
No hace falta tocar nada más.

## 🚀 Cómo verlo en mi computadora

```bash
npm install     # solo la primera vez
npm run dev     # abre http://localhost:5173
```

## 📦 Publicar cambios

Solo haz push a la rama `main`. GitHub Actions construye y publica el sitio
automáticamente.

```bash
git add .
git commit -m "Actualizo mi CV"
git push
```

## 🛠️ Tecnología

- React + Vite
- CSS puro (tema oscuro)
- GitHub Pages + GitHub Actions
